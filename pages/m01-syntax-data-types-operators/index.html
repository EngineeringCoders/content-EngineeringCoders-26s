1️⃣ Variables & Assignment

A variable in Python is a label (name) that points to an object in memory. You're not declaring memory — Python's garbage collector manages that.

Python
12345678910111213141516
# Simple assignment
name = "Praveen"
age = 28
height = 5.9

# Multiple assignment on one line
x = y = z = 0

# Tuple unpacking (swap without temp variable!)
a, b = 10, 20
a, b = b, a       # a=20, b=10 now — very Pythonic

# Augmented assignment
score = 100
score += 10       # score is now 110
score *= 2        # score is now 220

🔍 What id() tells you

Python
12345678910
x = 1000
y = 1000
print(id(x), id(y))     # Different ids — different objects
print(x == y)            # True  (same value)
print(x is y)            # False (different objects)

# ⚠️ Python caches small integers (-5 to 256)
a = 5
b = 5
print(a is b)            # True — same cached object!

💡 Tip: Use == to compare values. Use is only to compare with None, True, False — never for general equality.

Python has three numeric types built-in:

Type Example Use case
int 42, -7, 1_000_000 Whole numbers, no size limit
float 3.14, -0.001, 1.5e10 Decimals, IEEE 754
complex 3+4j, complex(3, 4) Signal processing, math
Python

# complex

print(c.real)    # 3.0
print(c.imag)    # 4.0
print(abs(c))    # 5.0  (magnitude = √(3²+4²))

Python
12345678910111213141516171819202122

# Debugging with = (Python 3.8+) ← very useful!
x = 42
print(f"{x = }")     # x = 42  (prints name AND value)

# Nested expressions
data = [10, 20, 30]
print(f"Sum: {sum(data)}, Max: {max(data)}")

Identity & Membership

Python
12345678910111213141516
# Identity
x = [1, 2, 3]
y = x           # y points to SAME list
z = [1, 2, 3]   # z is a DIFFERENT list with same values

print(x is y)   # True  — same object
print(x is z)   # False — different objects
print(x == z)   # True  — same values

# Membership
fruits = ["apple", "banana", "mango"]
print("apple" in fruits)       # True
print("grape" not in fruits)   # True

# Works on strings too
print("Py" in "Python")        # True

Bitwise (Important for AI/ML!)

Python
123456789
a = 0b1010    # 10
b = 0b1100    # 12

print(bin(a & b))   # 0b1000 = 8   AND  (both bits 1)
print(bin(a | b))   # 0b1110 = 14  OR   (either bit 1)
print(bin(a ^ b))   # 0b0110 = 6   XOR  (exactly one bit 1)
print(bin(~a))      # -11          NOT  (flip all bits)
print(bin(a << 1))  # 0b10100 = 20 left shift (multiply by 2)
print(bin(a >> 1))  # 0b101  = 5   right shift (divide by 2)

💡 Real-world use: Bitwise ops are used in image processing, ML flags, permissions systems, and low-level optimization.

6️⃣ The Walrus Operator := (Python 3.8+)

The walrus operator assigns AND returns a value in one expression. It's named after a walrus because := looks like walrus eyes.

Python
12345678910111213141516171819202122
# Old way — read data, check length separately
data = [1, 2, 3, 4, 5]
n = len(data)
if n > 3:
    print(f"Long list: {n} items")

# New way — assign inside the condition
if (n := len(data)) > 3:
    print(f"Long list: {n} items")  # n is available here too!

# ✨ Best use case: while loops that process and check
import random

while (value := random.randint(1, 10)) != 5:
    print(f"Got {value}, trying again...")
print(f"Finally got 5!")

# Another great use: regex matching
import re
text = "Order #12345 is ready"
if match := re.search(r"#(\d+)", text):
    print(f"Order number: {match.group(1)}")   # 12345

⚠️ Gotcha: Don't overuse walrus. Use it when the assigned variable is needed both in the condition and the body. If you only need it in the body, a regular assignment is cleaner.

7️⃣ Type Conversion

Python
123456789101112131415
# Implicit (Python does it for you)
result = 5 + 2.0    # int + float = float (3.0)
print(type(result)) # <class 'float'>

# Explicit (you do it intentionally)
print(int("42"))        # 42
print(int(3.9))         # 3  ← truncates, does NOT round
print(float("3.14"))    # 3.14
print(str(100))         # "100"
print(bool(0))          # False
print(list("abc"))      # ['a', 'b', 'c']

# int with base
print(int("1010", 2))   # 10 (binary to decimal)
print(int("FF", 16))    # 255 (hex to decimal)
